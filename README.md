# Playwright Automation Testing - Demo Project

This project demonstrates automated end-to-end testing using Playwright with TypeScript.

Features:
- Page Object Model (POM)
- Test reporting
- CI with github

## Project Structure

.
├── tests/            # Test cases
├── pages/            # Page Object Models
├── utils/            # Helper functions
├── playwright.config.ts
└── package.json
